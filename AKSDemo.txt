
AKS Learning
=============================

This is an excellent AKS learning project because you can learn:

Containers
Docker Engine / Podman
Container Registry
Kubernetes basics
Pods
Deployments
Services
Kubelet
Containerd
Kubectl
AKS
Internal service-to-service communication
Scaling
Rolling updates

--Big Picture
.NET Application
       ↓
Docker Image
       ↓
Azure Container Registry (ACR)
       ↓
AKS Deployment
       ↓
Pod
       ↓
Service
       ↓
Load Balancer
